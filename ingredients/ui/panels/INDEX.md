# Panel Ingredients

Major workspace panels that compose into project and settings windows.

| File | Description |
|------|-------------|
| [ai-settings-panel.md](ai-settings-panel.md) | Settings panel for AI/LLM provider configuration with multi-provider support |
| [code-editor-pane.md](code-editor-pane.md) | Text editor pane for source code with syntax highlighting, line numbers, minimap, dirty tracking, and auto-save |
| [debug-panel.md](debug-panel.md) | Debug-only panel for feature flag overrides, analytics monitoring, and runtime configuration inspection |
| [file-tree-browser.md](file-tree-browser.md) | Hierarchical file browser with lazy loading, git status badges, ignore patterns, and file-type icons |
| [inspector-panel.md](inspector-panel.md) | Right-side sliding panel that shows metadata for the currently selected item in the workspace |
| [terminal-pane.md](terminal-pane.md) | Multi-session terminal pane with PTY-backed shell sessions, session sidebar, and terminal profiles |
