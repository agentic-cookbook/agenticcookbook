# Toolbar

## Overview

Formatting toolbar for the document editor. Provides bold, italic, underline, heading, and list controls.

## Requirements

**toolbar-toggle**: Each formatting button MUST toggle its state to reflect current selection formatting.

**toolbar-disable**: Toolbar buttons MUST be disabled when no document is open.
