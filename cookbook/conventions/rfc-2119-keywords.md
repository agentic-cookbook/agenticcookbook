# RFC 2119 Keywords

Use RFC 2119 keywords for all behavioral requirements:

- **MUST** / **MUST NOT**: Absolute requirements. Every MUST requires at least one test vector.
- **SHOULD** / **SHOULD NOT**: Recommendations.
- **MAY**: Optional behavior.
