Proposed Slide Outline

~15 slides, technical audience, tight narrative

Act 1: The Shift (3 slides)

1

Title Slide

The Agentic Cookbook — branding, tagline

2

Cross-Platform Is Dead

The middleware tax: React Native, Flutter, Xamarin — write once through an abstraction layer. Made sense when humans were the bottleneck.

3

Multi-Platform Is Native

Agents don't need middleware. Given the right constraints, they write native code for 1 or 20 platforms simultaneously. Go straight to the best API.

Act 2: The Cookbook (5 slides)

4

What Is the Agentic Cookbook?

A structured knowledge base — principles, guidelines, recipes, compliance — consumed directly by AI agents. The "write once" is now the constraints, not the code.

5

Philosophy: Constrain, Don't Abstract

18 engineering principles, 88 guidelines organized by topic, 81 compliance checks. Define what "trusted" means across 14 dimensions.

6

Recipes: The New "Write Once"

27 recipes that define UI, infrastructure, behavior. A recipe constrains the agent — platform-specific code is the output. Show a recipe example.

7

How It All Connects

Diagram: recipes → compliance → guidelines → principles. Three-layer architecture. Everything cross-references.

8

Projects: Consuming the Cookbook

One rule file (cookbook.md), /install-cookbook onboarding. No tiers — everyone gets everything. How a project wires in.

Act 3: The Interview System (4 slides)

9

The Problem: Scoping

Before you build, you need to know what to build. Product discovery is still a human bottleneck — agents can help here too.

10

Multi-Agent Interview Team

Architecture diagram: meeting leader → transcript analyzer → specialist interviewers + analysts. Hub-and-spoke model. 12 domain + 6 platform specialists.

11

The Interview Flow

Structured → exploratory → discover new knowns → back to structured. Living checklist. Resume across sessions.

12

Specialist Intersection

Power is in the cross: domain knowledge × platform knowledge. Security specialist meets iOS specialist → authentication questions you wouldn't think to ask.

Act 4: The Full Picture (3 slides)

13

How They Fit Together

Interview → transcripts + analysis → recipes constrain → agent builds native for each platform. End-to-end diagram.

14

Using It: The Workflow

Clone repos → /install-cookbook → /interview → plan → build. Concrete steps a developer follows.

15

Closing

Links, repos, getting started. The middleware is dead — long live the recipe.
